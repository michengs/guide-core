# guide-core

稍微在[hsdn]的tera-guide-core 基础上简单添加了一些有趣的code

The guides based on this module can be found here: https://github.com/hsdn/tera-guide

下载解压放入 proxy/mods文件夹目录注意重命名为:guide-core

## Basic features

*为您最喜欢的地下城创建自定义指南的完整工具。

*支持添加语言翻译的功能。

*内置GUI支持所有指南设置。

*支持语音通知(使用Windows TTS模块)

*灵活和强大的功能，脚本指南。

*根据游戏客户端文件自动生成地下城列表。

*使用现代JavaScript提高性能。

*使用nodejs EventEmitter来处理所有的事件和钩子。
